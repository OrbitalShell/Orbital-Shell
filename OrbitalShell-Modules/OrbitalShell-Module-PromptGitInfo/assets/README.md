<b>Orbital Shell</b> is a multi-plateform (**windows, linux, macos**) command shell, inspired by <b><i>bash</i></b> and POSIX recommendations. Developed using **C# 8, .NET Core 3.1/Net 5 and .NET Standard 2.1**
